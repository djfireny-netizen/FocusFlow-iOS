## 🎯 快速准备截图指南

### 步骤 1: 使用模拟器截取

打开终端运行：

```bash
# 1. 启动 iPad Pro 12.9" 模拟器
xcrun simctl boot "iPad Pro 12.9-inch (6th generation)"

# 2. 在 Xcode 中运行 App 到该模拟器
# 选择设备: iPad Pro 12.9-inch (6th generation)
# 按 ⌘ + R 运行

# 3. 截图
# 在模拟器中按 ⌘ + S 截图
# 截图保存在桌面
```

## ⚠️ 常见问题

### Q: 必须有 iPad 截图吗？
**A**: 是的，App Store Connect 要求上传所有支持设备类型的截图。由于你的应用支持 iPad，必须提供 iPad 截图。

### Q: 截图尺寸不对怎么办？
**A**: 必须使用准确的分辨率。建议使用模拟器截图，或用手机截图后调整尺寸。

### Q: 版权信息填什么？
**A**: 填写 `© 2026 FocusFlow Team` 或 `© 2026 你的名字`
